# Guide utilisateur

## Pages principales
- **Connexion** : accès sécurisé à la plateforme.
- **Dashboard** : KPI et activité récente.
- **Tickets** : recherche, filtrage, consultation.
- **Ajouter** : création manuelle avec prévisualisation de l'analyse.
- **Import** : import CSV/JSON en lot.
- **Statistiques** : graphiques et imports récents.
- **Administration** : utilisateurs, paramètres, audit.

## Parcours type
1. Se connecter en administrateur.
2. Consulter les KPI du dashboard.
3. Ajouter un ticket ou en importer plusieurs.
4. Ouvrir le détail d'un ticket pour examiner le scoring.
5. Corriger si nécessaire la catégorie, la priorité ou l'urgence.
6. Consulter les logs et statistiques.
